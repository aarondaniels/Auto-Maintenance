# auto_maint_client

A standalone Flutter app for tracking vehicle maintenance locally on a device.

## Getting Started

This app stores vehicle, fillup, service, reminders, and stats data on-device.
No login or central backend is required.

### Run locally

```bash
cd client
flutter pub get
flutter run
```

## Notes

- The app persists data in the device's documents folder.
- No remote server or authentication is needed.
